# Distrilacteos Las Cabañuelas - Sistema Web

Sistema web para distribuidora mayorista de productos alimenticios ubicada en Bogotá, Colombia.

## Descripción

Plataforma web para Distrilacteos Las Cabañuelas que permite a clientes conocer el negocio y explorar el catálogo de productos. Incluye panel administrativo para gestión interna.

## Tecnologías

- HTML5, CSS3, JavaScript
- Node.js + Express
- MySQL

## Módulos

- Página pública: inicio, catálogo, detalle de producto, contacto
- Panel administrativo: CRUD de productos y categorías
